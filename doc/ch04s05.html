Using the JSAPResult

The result of JSAP.parse() is a JSAPResult object. This contains all of the values for each command line parameter. There are a large number of accessor methods available for reading the values: getString(), getBoolean(), getInteger(), getInetAddress(), getColor(), ...far too many to include here, including Iterators and arrays for reading multivalue parameters. The javadocs are pretty useful for this class.
